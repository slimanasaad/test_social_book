Project Scope;
Post creation interface for multiple users create posts such as (video, audio, image) files.


Project Technology Stack;
1-Python
2-Django
3-Sqlite3


Project Setup;

1-Install Python from https://www.python.org/ftp/python/3.12.2/python-3.12.2-amd64.exe

2-Open Command Prompt

3-Locate to the manage.py file of the project folder

4-run the command "pip install -r requirements.txt"

5-run the server by command "python manage.py runserver"

6-you will get an IP as http://127.0.0.1:8000/ visit this link on browser

7-to stop server press ctrl+c in the command prompt
